# Ave's Adventures Website

Vite + React + Tailwind mockup for the Ave's Adventures franchise site.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Where to edit content

Edit featured books, games, and character cards here:

```bash
src/data/siteData.js
```

The hero carousel is data-driven. Change the active books by editing `featuredBooks`.

## Logo assets

- Transparent PNG: `src/assets/aves-adventures-logo-transparent.png`
- SVG approximation: `src/assets/aves-adventures-logo.svg`

## Notes

This is a frontend mockup scaffold. The game buttons are ready to connect to future mini-games.
